## Snabbreferens

Fyra principer för nästa pass:

1. **Rå röst före prydlighet** — putsa språket utan att göra berättaren vuxnare.
2. **Ren svensk publikversion** — översätt engelska repliker och prosastycken; låt situationen bära språkförbistringen.
3. **Skam utan facit** — låt moraliskt obehag verka i scenen, inte i förklarande eftertext.
4. **Svensk muntlighet** — hellre levande, lite kantig prosa än korrekt men död prosa.
