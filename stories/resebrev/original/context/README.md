# Kontextmaterial

Här kan material som hör till resebrevens receptionshistoria och efterhandsförståelse ligga, utan att blandas ihop med själva originalbreven.

Artikelbilderna om Stora Sparrepriset ligger i `../articles/`.

Rå OCR ligger i `../articles/ocr/`:

- `IMG_0155.ocr.txt`
- `IMG_0156.ocr.txt`
- `IMG_0157.ocr.txt`
- `articles.ocr.combined.md`

Detta material ska behandlas som kontext för tolkning, röst och dramaturgi. Det behöver inte automatiskt bli scenmaterial i slutmanuset.
