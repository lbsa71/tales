# Redaktionellt Arbetslager

Detta lager styr upp arbetet med `Resebrev` utan att ändra originalmaterialet.

## Lager

- `../original/`: orörbart källmaterial från extraktionen.
- `../INTERVJUANTECKNINGAR.md`: författarsamtal, minnesmaterial och tematiska hypoteser.
- `../STRATEGISK_PLAN.md`: övergripande målbild och arbetsfaser.
- `MATERIALKARTA.md`: redaktionell läsning av materialet som helhet och per större rörelse.
- `KAPITELREGISTER.md`: praktiskt register över alla 25 brev med preliminär funktion och åtgärd.
- `GROV_OUTLINE.md`: föreslagen kapitelbåge, nyckelscener, stil och emotionell utveckling.

## Arbetsprincip

Nästa steg är inte språklig putsning, utan urval och disposition. Varje brev ska först få en funktion: oumbärligt kapitel, komprimerbart kapitel, fragment, kontextmaterial eller strykbar transport.
