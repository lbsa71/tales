# Example Draft

## Purpose

This file will hold short calibration passages for the final prose style. It is intentionally not filled with finished prose yet, because the project should first prove its chapter architecture through selection and cutting.

## Calibration Targets

Future examples should cover:

1. A lightly edited early comic passage.
2. A preserved moral-friction passage.
3. A 2026-adjacent bridge, if visible 2026 commentary is used.
4. A chapter ending that lets the scene resonate without explanation.

## Current Rule

Do not invent the finished voice before the raw manuscript has been cut.
