## Snabbreferens

Fyra principer att hålla levande under översättningen:

1. **Konsekvens före variation** — samma kärnterm genom hela verket
2. **Kropp före begrepp** — kroppstillstånd, doft, tryck, rytm; aldrig abstrakta etiketter
3. **Lyrisk återhållsamhet** — enkel syntax, hög precision, inga "smarta" formuleringar
4. **Var konkret, aldrig förklarande** — visa händelsen, förklara aldrig vad den "betyder"

Fullständig ordlista, röstkompass och kvalitetskontroll finns i [TRANSLATION_PLAN.md](../TRANSLATION_PLAN.md).
