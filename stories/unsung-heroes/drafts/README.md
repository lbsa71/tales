## Writing Workflow

### Recommended Approach (from IMPLEMENTATION_STRATEGY.md)

**Hybrid Method**:
1. Start with Chapter 7 (humans rising) - most familiar territory
2. Write Chapter 9 (narrator reveal) - anchor the voice
3. Fill in earlier rungs (Chapters 1-6) - build foundation
4. Write Chapter 8 (transition) - connect humans to narrator
5. Write Chapter 10 (successor) - complete the loop
6. Review and connect everything

### How to Use Templates

1. Copy a template file to create your working draft:
   ```bash
   cp rung_07_humans_rise_template.md rung_07_humans_rise_draft.md
   ```

2. Work in the `_draft.md` file (which is gitignored)

3. Reference planning artifacts as you write:
   - NARRATOR_VOICE.md
   - STYLE_GUIDE.md
   - THEME_TRACKER.md
   - EMOTIONAL_ARC.md
   - RUNG_DOSSIERS.md

4. Track your progress (optional, create your own tracking file)

### Daily Practice
1. Review previous day's work (5-10 min)
2. Check relevant artifacts (voice, themes, etc.)
3. Write new material (30-90 min)
4. Light edit of new material (10-15 min)

**Target**: 300-500 words per session

## Drafting Guidelines

### DO
- ✅ Write quickly, edit later
- ✅ Refer to NARRATOR_VOICE.md for consistency
- ✅ Keep THEME_TRACKER.md open for motifs
- ✅ Use STYLE_GUIDE.md example sentences
- ✅ Let yourself explore tangents
- ✅ Mark unclear sections with [TODO: clarify X]

### DON'T
- ❌ Edit while drafting
- ❌ Worry about perfect word choice yet
- ❌ Get stuck on one section too long
- ❌ Delete anything (move to "cuts" section)
- ❌ Compare to published works
- ❌ Obsess over transitions
